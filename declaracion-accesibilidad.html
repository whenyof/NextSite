Compromiso con la accesibilidad web universal

Última actualización: 19 de diciembre de 2024

Nuestro Compromiso

NextSite se compromete a garantizar la accesibilidad digital para todas las personas, independientemente de sus capacidades o limitaciones. Creemos que la tecnología web debe ser inclusiva y estar disponible para todos.

Nivel de Conformidad: AA (WCAG 2.1)

1. Estándares de Accesibilidad

Este sitio web cumple con los siguientes estándares internacionales de accesibilidad:

  • WCAG 2.1 (Web Content Accessibility Guidelines): Nivel AA
  • EN 301 549: Estándar europeo de accesibilidad
  • Ley 51/2003: Ley española de igualdad de oportunidades
  • Real Decreto 1112/2018: Accesibilidad de sitios web y aplicaciones móviles

2. Características de Accesibilidad Implementadas

Navegación por Teclado

Todas las funciones del sitio son accesibles mediante navegación por teclado. Use Tab para navegar y Enter para activar enlaces y botones.

Lectores de Pantalla

Compatibilidad completa con lectores de pantalla como NVDA, JAWS y VoiceOver. Estructura semántica HTML5.

Texto Escalable

El texto puede ampliarse hasta 200% sin pérdida de funcionalidad. Soporte para zoom del navegador.

Contraste de Colores

Relación de contraste mínimo de 4.5:1 para texto normal y 3:1 para texto grande según WCAG 2.1.

Imágenes Accesibles

Todas las imágenes incluyen texto alternativo descriptivo (alt text) para usuarios con discapacidad visual.

Diseño Responsivo

Adaptación automática a diferentes tamaños de pantalla y dispositivos móviles.

4. Tecnologías de Asistencia Compatibles

Este sitio web es compatible con las siguientes tecnologías de asistencia:

4.1. Lectores de Pantalla

  • NVDA (NonVisual Desktop Access): Gratuito para Windows
  • JAWS (Job Access With Speech): Comercial para Windows
  • VoiceOver: Integrado en macOS e iOS
  • TalkBack: Integrado en Android

4.2. Software de Ampliación

  • ZoomText: Ampliador de pantalla comercial
  • MAGic: Ampliador de pantalla comercial
  • Lupa de Windows: Herramienta integrada de Windows

4.3. Software de Reconocimiento de Voz

  • Dragon NaturallySpeaking: Control por voz
  • Voice Control (macOS): Control por voz integrado
  • Dictado de Windows: Herramienta integrada

5. Limitaciones y Excepciones

Aunque nos esforzamos por mantener la máxima accesibilidad, reconocemos las siguientes limitaciones:

5.1. Contenido de Terceros

  • Google Maps embebido puede no ser completamente accesible
  • Algunos servicios de análisis pueden no cumplir estándares AA
  • Contenido generado por usuarios puede no seguir nuestras pautas

5.2. Contenido Dinámico

  • Animaciones CSS pueden no ser compatibles con preferencias de movimiento reducido
  • Contenido cargado dinámicamente puede requerir revisión adicional

6. Mejoras Futuras

Estamos trabajando continuamente para mejorar la accesibilidad de nuestro sitio web:

  • Implementación de más opciones de personalización
  • Mejora del soporte para navegación por voz
  • Optimización para dispositivos de entrada alternativos
  • Pruebas regulares con usuarios reales

7. Contacto y Retroalimentación

Valoramos sus comentarios sobre la accesibilidad de nuestro sitio web. Si encuentra barreras de accesibilidad o tiene sugerencias de mejora:

7.1. Formas de Contacto

  • Email: info@nextsite.es
  • Asunto: "Accesibilidad Web"
  • Formulario de contacto: Disponible en la página principal

7.2. Información a Incluir

Al reportar problemas de accesibilidad, por favor incluya:

  • URL específica donde encontró el problema
  • Descripción detallada del problema
  • Tecnología de asistencia que está utilizando
  • Sistema operativo y navegador

8. Proceso de Resolución

Nos comprometemos a responder a todas las consultas de accesibilidad dentro de 5 días hábiles y a implementar soluciones en un plazo razonable.
